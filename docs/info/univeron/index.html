The origin of univeron

Univeron = Universe + Neuron

中文名为宇宙神经失常中心

嗯，是非常有创造性的名字呢。这就是我们的神经科学交流会 [1]。

这一个小小的社区是我、灯灯、钟钟还有胡老师共同创立的，在大家（尤其是琦琦）的帮助下，逐渐变得小有规模。这里有生物医学心理学计算机各个学科的朋友们，听上去好像就是能够搞什么大事情一样🙌。

因为热爱neuroscience我们聚集在一起。不过，这里不仅是学术讨论的地方——正如灯灯所说：

我们不仅仅是学术的人，同时也是属于这个社会的。在思考怎么打病毒怎么构质粒的同时，也应该去思考这个社会究竟是什么样；思考我们应该如何去认知、接纳和融入这个社会，并且在其中做自己觉得合乎良心的事情。

这还蛮重要的。

如果感兴趣的话，欢迎联系我们加入其中👉👈
